# Tailwind CSS Learning Project

This repository contains my practice work and mini-projects created while learning **Tailwind CSS v4**. It covers the fundamentals of building responsive and modern user interfaces using Tailwind's utility-first approach.

## 🚀 Features

- Responsive layouts
- Flexbox and Grid
- Typography utilities
- Color utilities
- Spacing and sizing
- Hover and focus effects
- Custom theme configuration
- Custom fonts
- Responsive navigation menu
- Tailwind CSS v4 utilities

- JavaScript

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Tailwind-css.git
```

Go to the project directory:

```bash
cd Tailwind-css
```

Install dependencies:

```bash
npm install
```

Start the Tailwind CSS compiler:

```bash
npm run build-css
```

Open `public/index.html` in your browser.
